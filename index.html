Hatena (はてな)

日本の代表的なソーシャルブックマークサービス。ブログ、ブックマーク、質問サイトなど多様なサービスを提供。動的なコンテンツと情報共有の中心的プラットフォーム。

  • Visit Hatena

GitHub

開発者向けの代表的なコード共有プラットフォーム。オープンソースプロジェクトの管理、バージョン管理、協働開発のためのツール群を提供。

  • Visit GitHub

Qiita

プログラマのための技術情報共有サービス。技術記事の投稿・閲覧、知識の蓄積と共有により、開発者コミュニティの発展に貢献。

  • Visit Qiita

note

クリエイターのためのコンテンツ配信プラットフォーム。記事、音声、動画など多様な形式でコンテンツを共有し、クリエイターの収益化をサポート。
